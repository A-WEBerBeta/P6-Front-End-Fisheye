# FishEye - Plateforme de photographes accessible

Projet réalisé dans le cadre de la formation Développeur d'application JavaScript React chez OpenClassrooms.

## Objectif
Refondre une plateforme web de photographes en améliorant l'accessibilité et l'expérience utilisateur, tout en respectant les bonnes pratiques du développement frontend.

## Fonctionnalités
- Affichage dynamique des photographes et de leurs médias
- Navigation clavier complète
- Amélioration du balisage sémantique
- Gestion des modales accessibles
- Respect des recommandations WCAG

## Technologies
- HTML
- CSS
- JavaScript

## Accessibilité
- Navigation au clavier
- Gestion du focus
- Contrastes adaptés
- Balises sémantiques
